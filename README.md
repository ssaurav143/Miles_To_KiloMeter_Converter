# 🚗 Miles to Kilometer Converter

A simple GUI based Miles to Kilometer Converter built using Python Tkinter.

The application allows users to enter distance in miles and instantly convert it into kilometers with accurate decimal formatting.

---

## 🎮 Features

- Simple and clean graphical interface  
- Converts miles to kilometers instantly  
- Button based conversion system  
- Decimal precision formatting  
- Built using Python Tkinter  

---

## 🧠 How It Works

- User enters distance in miles  
- Clicking the "Calculate" button performs the conversion  
- Result is displayed in kilometers  

Conversion Formula:

1 Mile = 1.60934 Kilometers
